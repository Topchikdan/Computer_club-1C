# Учёт компьютерного клуба 1С (Club-1C)

Конфигурация 1С для автоматизации компьютерного/игрового клуба: учёт залов и компьютеров, бронирование мест, продажа товаров и услуг бара, складской учёт и работа с клиентами.

## О проекте

Конфигурация объединяет несколько контуров автоматизации клуба:

- **Бронирование и эксплуатация** — учёт залов, компьютеров, игр и бронирования мест клиентами.
- **Торговля и бар** — продажа и закупка товаров/услуг (включая ассортимент бара), складской учёт остатков и списание товаров.
- **Клиенты** — карточки клиентов с поддержкой VIP-скидок.
- **Кадры** — базовый кадровый учёт сотрудников клуба (приём на работу, увольнение, кадровые перемещения).
- **Аналитика** — отчёты по выручке и состоянию склада.

Исходники выгружены штатным механизмом платформы (**Конфигурация → Выгрузить конфигурацию в файлы...**) в текстовом/XML-виде, что позволяет вести историю изменений в git без хранилища конфигурации (актуально для базовых версий 1С, где хранилище недоступно).

## Состав конфигурации

### Справочники

| Справочник | Назначение |
|---|---|
| **Залы** | Игровые залы клуба |
| **Компьютеры** | Учёт компьютеров клуба (привязка к залам, характеристики) |
| **Игры** | Каталог игр, доступных в клубе |
| **Клиенты** | Карточки клиентов, в т.ч. с признаком VIP |
| **ТоварыИУслуги** | Номенклатура товаров и услуг (включая ассортимент бара) |
| **ВидТоваровИУслуг** | Классификация товаров и услуг |
| **Сотрудники** | Карточки сотрудников клуба |
| **Должности** | Перечень должностей |
| **Отделы** | Структурные подразделения |
| **ПолСотрудника** | Справочник пола сотрудника |

## Требования

- Платформа 1С:Предприятие 8.3 (для открытия и редактирования конфигурации в Конфигураторе)
- [OneScript](https://oscript.io) — при использовании пакетного менеджера `opm` и вспомогательных утилит

## Как обновить репозиторий после изменений в конфигурации

1. Откройте базу в режиме **Конфигуратор**.
2. Выполните **Конфигурация → Выгрузить конфигурацию в файлы...**, указав эту же папку.
3. Зафиксируйте изменения:

```bash
git add .
git commit -m "Описание изменений"
git push
```

## Как развернуть конфигурацию из репозитория

1. Создайте новую информационную базу в 1С (или используйте существующую).
2. В Конфигураторе выполните **Конфигурация → Загрузить конфигурацию из файлов...**, указав путь к этой папке.
3. Обновите конфигурацию базы данных (**F7**).
